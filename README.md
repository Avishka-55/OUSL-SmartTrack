# 🎓 Smart Tracker — OUSL Student Suite

**Smart Tracker** is a centralized web portal built to help Open University of Sri Lanka (OUSL) students easily access essential academic tools. It links together various calculators and planners to streamline the student experience, from planning semesters and calculating fees to tracking GPAs and exam targets.

This project is built with **React** and **Vite** for a fast, modern web experience and acts as a central hub for individual OUSL tools.

## 🚀 Running This Hub Locally

If you'd like to run the Smart Tracker hub on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Avishka-55/OUSL-SmartTrack.git
   cd OUSL-SmartTrack
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the `localhost` URL provided in the terminal.
